# MACE Windowとは

MultiAbilityCartoonExpressWindow

# 開発環境
* VisualStudio2019
* C#
* .NET Framework4.7.2

# 操作説明

| 操作 | 説明 |
| ------------- | ------------- |
| 左クリック+移動  | ウィンドウの移動  |
| 左クリック  | 画像の選択  |
| 左ダブルクリック  | 全画面切り替え  |
| 中クリック  | アプリケーションの終了  |
| スペースキー  | 最小化  |
| 左キー  | 前の画像  |
| 右キー  | 次の画像  |
